# Hajj & Umrah 3D Guide

Interactive English and Gujarati Hajj and Umrah learning website based on the uploaded authentic guide.

## Live website

https://sahilalad.github.io/hajj-umrah-3d-guide/

## Current features

- Interactive 3D-inspired Kaaba experience
- English and Gujarati language switching
- Step-by-step Umrah journey
- Day-by-day Hajj journey
- Arabic duas with English and Gujarati meanings
- Audio using the device speech system
- Local progress tracking without an account
- Mobile, tablet and desktop responsive design

## Project status

This is the first deployable build. The full page-by-page book transcription, translation and reference verification will be added progressively.
